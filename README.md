# The Most Annoying Alarm Clock

This alarm clock will use (in order of severity):

- Voice-lines from my friends telling you to get up
- A bright two-servo-aimed flashlight (which will use Camera Vision to aim)
- A water gun (which will also use Camera Vision to aim)
- Boxing Glove on a linear actuator (lol)

The project will also use camera vision to detect where my face is (as to aim the water/flashlight) and whether I'm still in bed or not (as to keep the alarm clock going or stop).
